# UNIX and Work with Genomics Data

Autor: Zuzana Sevcovicova

## Overview

This repository contains materials and exercises from the Next-Generation Sequencing (NGS) course, focusing on UNIX command-line tools and genomics data analysis. The course content follows the curriculum available at [NGS Course - Prague February 2026](https://ngs-course.readthedocs.io/en/praha-february-2026/index.html).

## About the Course

The NGS course is designed to teach participants how to:
- Work effectively with UNIX/Linux command-line interfaces
- Handle and process genomics data files
- Use bioinformatics tools for Next-Generation Sequencing data analysis
- Apply best practices in computational genomics workflows

## Course Topics

This repository covers essential topics including:

- **UNIX Fundamentals**
  - Command-line navigation and file management
  - Text processing and data manipulation
  - Shell scripting basics
  - Pipeline construction and automation

- **Genomics Data Handling**
  - Common genomics file formats (FASTA, FASTQ, SAM/BAM, VCF, GFF/GTF)
  - Quality control and preprocessing
  - Data filtering and transformation
  - Working with large-scale genomic datasets

- **Bioinformatics Workflows**
  - Sequence alignment and mapping
  - Variant calling and analysis
  - Gene expression analysis
  - Data visualization

## Repository Structure

This repository is organized to support hands-on learning through practical exercises and examples that complement the course materials.

## Getting Started

To work with the materials in this repository:

1. Clone this repository:
   ```bash
   git clone https://github.com/Sevcoviz/UNIX-and-work-with-genomics-data.git
   cd UNIX-and-work-with-genomics-data
   ```

2. Follow the course materials at the [NGS Course website](https://ngs-course.readthedocs.io/en/praha-february-2026/index.html)

3. Work through the exercises and examples provided
